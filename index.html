Detect and Correct: A Hybrid Feedback Framework for Mitigating Hallucinations in LLM-based Code Generation

Yinsheng Yao First Author
Supervised by Prof. Tianyi Zhang (Purdue University)
In Preparation 2025

Abstract & Motivation

In LLM-based code generation, models often produce "silent" logical errors—hallucinations that execute without crashing but yield incorrect results. Traditional Self-Debug methods rely solely on runtime tracebacks (e.g., TypeError), failing to diagnose these subtle flaws. This project introduces a Hybrid Feedback Framework that combines dynamic execution verification with a novel static Hallucination Detector, enabling models to identify and repair both runtime crashes and deep logical inconsistencies.

Key Results

  • Broad Robustness: Consistently outperformed standard baselines on HumanEval, MBPP, LiveCodeBench, and BigCodeBench across 5 LLMs.
  • Breaking Cognitive Fixation: Qualitative analysis confirms the framework disrupts repetitive error loops by providing novel attentional cues that force a re-evaluation of the core algorithm.
  • Capability Amplifier: Analysis revealed an inverted U-shaped gain, proving effectiveness at the "edge of capability" to break cognitive fixation loops.

Core Innovation

  • Synergistic Hybrid Feedback Unified repair loop integrating execution-based verification with model-based hallucination detection.
  • AST-Aware Detector Analyzes Abstract Syntax Trees and token log-probabilities to pinpoint logical flaws without tracebacks.
  • Error Summarization Processes raw tracebacks into actionable summaries to prevent context overflow.

MAVEN: Multi-Agent Verification and Elaboration Network

Yinsheng Yao First Author
Supervised by Prof. Dawei Cheng (Tongji University)
In Preparation 2025

Abstract & Motivation

Standard Large Language Models often suffer from "mode collapse" and hallucinations during complex reasoning. This project introduces MAVEN, a multi-agent framework designed to emulate System 2 (slow) thinking. By shifting the generation process from a linear trajectory to an iterative, adversarial workflow, MAVEN aims to maximize factual accuracy and logical consistency in complex Question Answering (QA) tasks without relying on external retrieval (RAG).

Key Results

  • Robust Performance: Outperformed standard Chain-of-Thought (CoT) and three collaborative baselines across the OpenBookQA and TruthfulQA benchmarks.
  • Quality Dominance: Achieved state-of-the-art scores across four critical dimensions: Justification Plausibility, Factual Accuracy, Comparative Analysis, and Argument Structure.
  • Enhanced Verifiability: Produced significantly more interpretable and verifiable reasoning chains through the framework's intrinsic adversarial scrutiny mechanism.

Core Innovation

  • Adversarial Deliberation "Skeptic-Responder-Judge" loop where agents generate counter-arguments to verify drafts.
  • Parametric Verification Treats internal parameters as a queryable database to isolate fact-checking.
  • Blackboard & Routing Shared workspace with dynamic routing resources (O(1) for simple vs O(N) for complex).
