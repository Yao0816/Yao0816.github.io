Selected Research Works

Detect and Correct: A Hybrid Feedback Framework for Mitigating Hallucinations in LLM-based Code Generation

Yinsheng Yao First Author
Supervised by Prof. Tianyi Zhang (Purdue University)
In Preparation 2025

Motivation & Insight

Standard self-debug methods rely on runtime tracebacks, failing to catch "silent" logical errors. Initial attempts using token confidence levels resulted in high false positives—mistaking valid grammar for errors. This project introduces a Hybrid Feedback Framework that shifts the paradigm from simple confidence scoring to semantic unit analysis, allowing the model to understand the logic behind its own mistakes.

Key Results

  • Semantic Precision: Reduced false positives by redefining probability aggregation methods to align with program logic logic.
  • Broad Robustness: Consistently outperformed standard baselines on HumanEval, MBPP, LiveCodeBench, and BigCodeBench.
  • Breaking Fixation: Disrupts repetitive error loops by providing novel attentional cues.

Core Innovation

  • Semantic Unit Detection Extends detection granularity from tokens to code fragments to reflect overall logic.
  • Hybrid Feedback Unified repair loop combining execution verification (Crash) and model-based detection (Logic).

MAVEN: Multi-Agent Verification and Elaboration Network

Yinsheng Yao First Author
Supervised by Prof. Dawei Cheng (Tongji University)
In Preparation 2025

Abstract & Engineering Logic

Standard LLMs often suffer from "mode collapse" during complex reasoning. MAVEN introduces an iterative, adversarial workflow to emulate System 2 thinking. Engineering Constraint: Observed initially that agents communicating via natural language or regex led to parsing failures and role confusion. I implemented a standardized JSON protocol and specific output constraints, treating the interaction as a structured system rather than open-ended dialogue.

Key Results

  • System Stability: Significantly reduced role redundancy and logical conflicts through strict functional division (Planning vs. Questioning).
  • Quality Dominance: Achieved state-of-the-art scores in Justification Plausibility and Factual Accuracy.

Core Innovation

  • Adversarial Deliberation "Skeptic-Responder-Judge" loop generating counter-arguments to verify drafts.
  • JSON-Based Communication Unified output format to facilitate structured transmission and prevent parsing errors.
